フリーBGMの置き場所
====================

このフォルダに「field.mp3」という名前でBGMファイルを置くと、
ゲームのBGMが自動でそのファイルに切り替わります（無い間は内蔵チップチューンが鳴る）。

おすすめのフリーBGMサイト（メイプル風ののどかな曲が見つかる）:
- PeriTune（ペリチューン） https://peritune.com/  … オーケストラ風・かわいい系が豊富。クレジット表記で無料
- DOVA-SYNDROME https://dova-s.jp/  … 国内最大級。「ほのぼの」「ファンタジー」で検索
- 魔王魂 https://maou.audio/  … ゲーム向け定番

※公開時はクレジット表記（作曲者名）をゲーム内かREADMEに入れること。
※将来マップごとに曲を変える場合は field.mp3 / forest.mp3 のように増やす予定。
