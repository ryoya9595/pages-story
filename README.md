# PAGESストーリー 〜描きかけの絵本の国〜

最大4人協力の探索型2DアクションRPG（スマホ＋PCブラウザ対応）。

- 企画書: `../企画書_PAGES_描きかけの絵本の国.md`
- 現在のフェーズ: **①技術検証プロト**（4人が同じマップを歩き回って互いに見える）

## 構成

| フォルダ | 役割 | 技術 |
|---|---|---|
| `server/` | ゲームサーバー（ルーム管理・状態同期） | Colyseus + TypeScript |
| `client/` | ゲーム画面 | Phaser 3 + Vite + TypeScript |

## 起動方法

```sh
# ターミナル1: サーバー（ws://localhost:2567）
cd server && npm install && npm run dev

# ターミナル2: クライアント（http://localhost:5173）
cd client && npm install && npm run dev
```

ブラウザで http://localhost:5173 を開く。タブを複数開けばマルチプレイ確認可能。
スマホ実機はPCと同じWi-Fiに繋いで `http://<PCのIPアドレス>:5173` を開く（バーチャルパッド表示）。

## 操作

- **PC**: ←→ / A・D 移動、↑ / W / スペース ジャンプ、J / X 攻撃、H 回復（パレットメイトのみ）
- **スマホ**: 左半分ドラッグ=移動、右下=とぶ、その左=こうげき、上=かいふく（パレットメイトのみ表示）

## 実装済みの仕様

- **同期**: 移動はクライアント主導20Hz送信＋他者補間。敵AI・HP・経験値・クエスト・回復はサーバー権威
- **戦闘**: ラクガキ敵（パトロール・タッチダメージ・6秒リスポーン）。死亡で3秒後もくじ広場復活（ロストなし）
- **経験値分配**: 1ダメージ以上で対象。貢献度50%＋レベル比50%＋人数ボーナス(2人+10%/3人+15%/4人+20%)
- **マップ**: ページ分割方式。もくじ広場(Lv1-2) ⇔ らくがきの森(Lv6-10ボス)。扉に重なると移動
- **セーブ**: 匿名ID+secret(localStorage)→`server/data/players.json`。15秒オートセーブ。右上「ひきつぎコード」でURLコピー→別端末引き継ぎ
- **あいことば部屋**: 同じあいことばを入れた人同士が同じ世界（最大4人）に入る。右上の「あいことば」ボタンで変更、`?room=◯◯` 付きURLでも指定可。デフォルトは「ひろば」
- **サーバー指定**: `?server=<ホスト名>` でwss接続先を指定可能（トンネル・本番用）。指定なしなら同ホストの:2567
- **クエスト**: q1〜q4チェーン（広場3体→広場5体→森3体→ボス）。報酬厚め、討伐貢献者全員が進行
- **職業**: 1次Lv10→2次Lv30→3次Lv60→4次Lv90の派生ツリー（全28職・後戻り不可）。`shared/jobs.ts` 参照
  - ✏️ エンピツ系（近接）: 手数の系譜（シャーペン剣士→ドラフター→線聖）／一撃の系譜（木炭剣士→デッサンマスター→グラファイトキング）
  - 🖍️ クレヨン系（タンク）: 鉄壁の系譜（パステルガード→オイルナイト→インパストキング）／攻守の系譜（ワックスブレイバー→メルトファイター→クレヨンブレイズ）
  - 🖋️ インク系（遠距離）: 連射の系譜（万年筆ガンナー→ボールペンスナイパー→筆神）／威力の系譜（墨スプラッシャー→墨絵師→水墨龍）
  - 🎨 パレット系（サポート）: 癒やしの系譜（水彩ヒーラー→フレスコセイント→レインボーマエストロ）／戦うサポートの系譜（アクリルペインター→ミューラリスト→マスターピース)
  - 系統で攻撃速度も変わる（手数型=連打、一撃型=重い）
- **回復**: 自然回復（被弾5秒後から4%/秒）＋パレットメイトのヒール
- プレイヤーカラー4色 = 鏡の大迷宮の4カービィへのオマージュ

## 開発用チート

サーバーを `ALLOW_DEV=1` で起動すると、ブラウザのコンソールから経験値を付与できる（バランステスト用）:

```js
__game.scene.keys.game.room.send("devGrantExp", { amount: 3000 }); // Lv10超まで一気に
```

公開時は ALLOW_DEV を外すこと。
