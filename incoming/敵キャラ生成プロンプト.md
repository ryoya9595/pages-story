# 敵キャラ生成プロンプト（ChatGPT用）

## 使い方
1. ChatGPTの**同じチャット内**で1体ずつ生成（スタイルが揃いやすい）
2. 1体目は「共通スタイル＋キャラ指定」をまるごと貼る
3. 2体目以降は「同じスタイルで、次は〜」とキャラ指定だけ貼る
4. 出てきたPNGを下のファイル名で保存して `pages-story/incoming/` に入れる
5. 透過にならなかったら白背景のままでOK（こっちで背景除去する）

---

## 1体目: マキマキ（保存名: makimaki.png）

```
2D横スクロールアクションゲーム用のモンスタースプライトを1体描いてください。

【スタイル指定】
- 丸っこくて目が大きい、2頭身のかわいいデフォルメモンスター
- 明るくはっきりした色、太くクリーンな輪郭線
- レトロゲーム風のドット絵（ピクセルアート）タッチ
- 横向き（左向き）、全身、1体のみ
- 背景は完全に透明（透過PNG）。影や地面は描かない

【キャラ指定】
名前「マキマキ」: 鉛筆の削りカスがくるくる巻いた殻になっているカタツムリ。
体はクリーム色でぷにぷに、殻は薄い木の色に茶色い縁取りの削りカス。
のんびりした、ちょっと眠そうなかわいい表情。
```
